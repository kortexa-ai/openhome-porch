# Contributing to Porch

Thanks for your interest in contributing! Porch is open to contributions from anyone.

## Getting Started

1. Fork this repo
2. Clone your fork
3. Create a branch: `git checkout -b my-feature`
4. Make your changes
5. Open a Pull Request against `main`

## Development

Porch is a native Swift macOS app. You'll need:

- Xcode 16+
- macOS 14.0+ (Sonoma)

Open the project in Xcode and build (Cmd+B). No additional dependencies required.

## Guidelines

- Keep it simple. Porch is meant to be lightweight.
- Follow existing code style (Swift conventions, 4-space indent).
- One PR per feature or fix.
- Include a clear description of what your PR does and why.
- Test on macOS before submitting.

## Reporting Issues

- Use [GitHub Issues](../../issues) for bugs and feature requests.
- Include your macOS version and steps to reproduce.

## Code of Conduct

Be kind. Be constructive. We're all here to build something useful.
